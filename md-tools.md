## Code & Data

An `inline` portion of code.

```java
PASTE_JAVA_CODE_HERE
```

```xml
PASTE_XML_DATA_HERE
```

```properties
PASTE_PROPERTIES_DATA_HERE
```

```yaml
PASTE_YAML_DATA_HERE
```

* Link with label: `[title](link)`

## Tasks

* [x] checked: `* [x]`

## Specials Characters

### Escape characters

``backtick `in backtick`` -> double first & last

> Markdown provides backslash escapes for the following characters:

```text
\   backslash
`   backtick
*   asterisk
_   underscore
{}  curly braces
[]  square brackets
()  parentheses
#   hash mark
+   plus sign
-   minus sign (hyphen)
.   dot
!   exclamation mark
```
